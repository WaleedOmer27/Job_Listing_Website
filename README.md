# 📌 Job Listing Website

A modern **Job Listing Web Application** built with **React, Vite, Tailwind CSS, and JSON Server (mock REST API)**.  
This project allows users to browse jobs, view details, and perform full CRUD operations (Create, Read, Update, Delete).

---

## 🚀 Live Demo

Add your deployed link here (Vercel / Netlify)


https://joblistingwebsite-delta.vercel.app/


---

## ✨ Features

- 🧾 View all job listings
- 🔍 Search and filter jobs
- ➕ Add new job listings
- ✏️ Edit existing jobs
- ❌ Delete jobs
- 📄 View detailed job page
- ⚡ Fast navigation using React Router
- 🎨 Responsive UI with Tailwind CSS

---

## 🛠️ Tech Stack

- React (Vite)
- React Router DOM
- Tailwind CSS
- JSON Server (Mock Backend)
- Fetch API (GET, POST, PUT, DELETE)

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/WaleedOmer27/Job-Listing-Website.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

🧪 JSON Server (Mock Backend)

If your project uses JSON Server:

Install globally (if needed):

```bash
npm install -g json-server
```

Run server:

```bash
json-server --watch db.json --port 8000
```

Your API will run at:

```bash
http://localhost:8000/jobs
```
